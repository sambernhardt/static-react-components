Static-ish React Components

How to replace static HTML elements with React components.


Intro

Let's say we want to replace a static HTML button with a componentized version.

Define your Button component as usual, but use the dangerouslySetInnerHTML prop to pass any children through to the Button's inner HTML:

              import React from 'react';
              import styled from 'styled-components';
      
              const Button = styled.div`
                  padding: 12px 16px;
                  border-radius: 4px;
                  background: black;
                  color: white;
                  cursor: pointer;
              `;
      
              export default ({ children }) => (
                  <Button
                      dangerouslySetInnerHTML={{ __html: children }}
                  />
              )

But instead of mounting an <App/> component to Index.js selects static HTML elements with the data attribute data-component.

              import Button from './components/Button';
                  document.querySelectorAll('[data-component="Button"]').forEach(button => {
                  ReactDOM.render(
                  <Button children={button.innerHTML} />
                  , button);
                  })

Then the bundled index.js file can be used in any non-react context:

              <span data-component="Button">
                    Button Label
                  </span>
                  <script src="react_bundle.js"></script>
              
            
              
                  <span data-component="Button">
                  <div class="sc-AxirZ bJCmFu">
                  Button Label
                  </div>
                  </span>
